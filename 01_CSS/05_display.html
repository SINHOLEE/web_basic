2. inline level element

a tag span tag

!!!중요 !!! inline은 한줄에 표시되며, width, height, margin-top, margin-bottom을 조절할 수 없다.

인라인에선, 마진 탑바텀을 줄 수 없다. 패딩으로는 상하좌우 조절 가능하지만, 패딩으로 공백을 조율할 경우, 다른 블럭과 겹치는 불상사가 생긴다.

3. inline-block

  1. inline 이지만 block에서 활용되는 속성을 활용할 수 있다.
  2. width, height, margin-top, margin-bottom 등과 같은 적용할 수 있다.
